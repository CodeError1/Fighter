إعدادات GitHub

بيانات الريبو اللي الموقع منشور عليه على GitHub Pages. الإعدادات دي بتتحفظ في المتصفح بتاعك بس.

محتاج تعمل GitHub Token مرة واحدة من هنا (Fine-grained token) وتديله صلاحية Read and write على Contents بس، لنفس الريبو ده. التوكن ده زي الباسوورد بتاعت الريبو، متشاركوش مع حد.

شعار تطبيق

اكتب كود المشروع اللي عايز تحدّث شعاره، وارفع الصورة. أي صورة هترفعها هتتحول تلقائي لمقاس مربع PNG.

إدارة تطبيقات المتجر

هنا بتتحكم في التطبيقات الظاهرة في المتجر (الاسم والوصف). كل تطبيق ليه كود بسيط بالإنجليزي (Project ID) بيربطه بملفات الأيقونة والـ APK بتاعته.

بيتم التحميل...

لو الكود جديد، دوس على "حفظ في المتجر" الأول، وبعدين ارفع شعاره من الكارت اللي فوق، وملف الـ APK بتاعه من GitHub Desktop زي الكارت اللي تحت.

ملف التطبيق (APK)

ملفات الـ APK كبيرة الحجم والرفع المباشر من المتصفح مش موثوق فيها، فبتترفع من GitHub Desktop بدل هنا.

1. افتح فولدر الريبو على جهازك.
2. حط ملف الـ APK في المسار المناسب:

للتطبيق الأساسي (app1): app-release.apk (في جذر الريبو)

لأي تطبيق تاني: projects/<كود المشروع>/app-release.apk

3. من GitHub Desktop: Commit ثم Push.
